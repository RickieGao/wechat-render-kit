# 新主题设计 Brief 模板

把这份 brief 填完后，交给 `/new-theme` 的 **fresh 路径**（或直接对话里逐项回答），让 theme-designer 为你生成一套全新的公众号主题。

> **用时估计**：约 2 分钟。全部字段都有说明，没把握的字段可以留空或写"不限"。

---

## 基本信息

**主题名** `主题名`
- 请填：kebab-case 小写短横线格式，例如 `mono-editorial`、`warm-ink`、`nordic-data`。这是主题文件名，不可含空格或中文。

**family** `fresh 全新设计 / inherited 继承你旧号 DNA`
- 请填：本模板适用 **fresh**（从零设计）。如果你想继承现有号风格，请走 `/new-theme` 的 inherited 路径，该路径不需要此模板。

**基调 tone** `humanist 偏感性书页感 / tech 偏理性几何感`
- 请填：内容以散文 / 回忆录 / 感性表达为主 → `humanist`；以数据复盘 / 工程博客 / 理性分析为主 → `tech`；两者兼有可以写"偏 humanist"或"偏 tech"。

---

## 视觉风格

**参考刊物 / 网站**
- 请填：你欣赏的长文排版，例如 `Stratechery`、`The New Yorker`、`Linear blog`、`Substack 某号`、`Notion Updates`。1–3 个即可；这是设计师找感觉的锚点，不是要求完全复制。

**字体倾向**
- 请填以下 3 项（不确定可写"随设计师"）：
  - 标题字体：`衬线` / `无衬线`
  - 正文字体：`衬线` / `无衬线`
  - 是否重用等宽 mono（SF Mono / Menlo 系）：`是` / `否` / `仅用于数据类段落`

**配色情绪**
- 请填：
  - 冷暖倾向：`暖（橘 / 米 / 土）` / `冷（蓝 / 灰 / 青）` / `中性`
  - 主强调色（用于 H2 前缀、`<strong>`、装饰线等）：可写具体色值如 `#D35400`，或色系描述如"深橄榄绿"
  - 页面背景：`白 #FFFFFF` / `米白 rgb(250,247,241)` / `其他（请描述）`

**装饰语言**
- 请填：
  - 整体风格：`几何线性（线、框、数字编号）` / `有机书页（曲线、留白、衬线装饰）` / `极简（无装饰，靠字重/间距区分层级）`
  - 可选装饰件（勾选你想要的）：
    - [ ] 首字下沉（lead paragraph 第一个字放大浮动）
    - [ ] 章节编号前缀（如 `01 /`、`I.`、`§`）
    - [ ] H2 装饰带或底边线
    - [ ] 大号引号装饰（`"` 或 `「」`）作 blockquote 背景
    - [ ] 章节分隔符自定义（替换 `<hr>`，如 `· · ·`、`* * *`、`§`）
    - [ ] 其他（请描述）：___

**信息密度**
- 请填：`紧凑（段间距小、字号较小、适合数据密集内容）` / `疏朗（大间距、大行高、适合长散文）`

## 固定覆盖清单（预填，无需修改）

以下两个清单是**所有主题必须覆盖**的 — theme-designer 会按此清单产出完整 HTML reference。无需修改，仅供确认。

### 全套 markdown 元素

`<h1>` `<h2>` `<h3>` `<p>` `<strong>` `<em>` `<a>` `<ul>` `<ol>` `<li>` `<blockquote>` `<table>`（含 thead/tbody）`<code>`（行内）`<pre><code>`（代码块）`<hr>` `<img>`

### 7 个结构性 primitives

1. `section-title` — 文章二级章节标题（H2），视觉骨架，反复出现
2. `inline-emoji` — 段首装饰图标位置
3. `image-caption` — 配图说明（画名 · 作者 · 年份三段信息）
4. `signature` — 文末个人签名（占位文字「作者」，勿填真名）
5. `ai-disclosure` — AI 工具透明披露卡
6. `quote-block` — 引用块
7. `sources` — 文末「想法出处」块（非正式参考文献，谦逊语气）

详细覆盖要求见 `docs/wechat-render/references/design-brief-fresh.md`。

## 填写示例

以下是一份填好的 brief，供参考：

```
主题名：mono-editorial
family：fresh
基调 tone：tech（偏理性）

参考刊物：Stratechery、Ribbon Farm

字体倾向：
  标题：无衬线
  正文：无衬线
  mono：是，用于元数据条和章节编号

配色情绪：
  冷暖：冷（深青 / 石板灰）
  主强调色：深青 rgb(6, 95, 70)
  背景：白 #FFFFFF

装饰语言：
  风格：几何线性
  勾选：章节编号前缀（01 /）、H2 装饰带

信息密度：紧凑
```
