## 7 个标志性组件

1. **小节标题** `| 文字`：32px 鲜艳蓝竖线 + 18px 黑色标题
2. **段首 emoji**：每个段落用 emoji 起首（🏋️ 🧪 🦠 🩻 🩺 🍔 🏸 😴 📋 🎯）
3. **画作引用**：白字灰底 `#888` 小标签，格式"画名 - 作者, 年份"
4. **结尾签名**：右对齐 GIF + 25% 宽的 `#1D1D1D` 细线
5. **AI 创作声明卡片**：`#ECEEF2` 底 + `#1B66B3` 圆形 🤖 + 同色标题
6. **红色引用块**：左侧 `#F03C16` 3px 竖线 + 14px 字
7. **想法出处块**：`#F4F4F4` 底 + 12px 字 + 1.8px 字距 + 红角标装饰 + `*注：` 谦逊语气
