# 风格 DNA 提取法

对着你已有的公众号文章，逐篇审视那些反复出现的视觉习惯——配色、标题装饰、段落节奏、签名形式——把它们归纳成一份可复用的「风格 DNA」。这份 DNA 就是 `/new-theme` classic 路径的输入：它会帮你生成一套在现有视觉风格基础上精细打磨的新主题，而不是推倒重来。

---

## 提取 4 个维度

对照你自己的公众号文章，逐项填写：

### 基础排版

| 项 | 你的值 |
|---|---|
| 正文色 | `#______` |
| 行高 | （例：`1.75`） |
| 字距 | （例：`0.5px`） |
| 字号 | （例：`16px`） |
| 段距 | （例：`margin-bottom: 12px`） |
| 对齐 | `justify` / `left` |
| 边距 | （例：`padding: 0 14px`） |

### 招牌色

| 用途 | 你的值 |
|---|---|
| 主强调色（招牌色） | `#______` |
| 次要文字色 | `#______` |
| 分割线色 | `#______` |
| 卡片底色（AI 声明 / 出处块 / 引用等） | `#______` |
| 其他特殊色（如角标、红线引用等） | `#______` |

### 标志性组件

逐一列出你文章里反复出现的视觉符号，例如：

- [ ] 小节标题样式（如：竖线 + 标题、数字前缀、罗马数字等）
- [ ] 段首装饰（如：emoji 锚点、首字下沉、横线引导）
- [ ] 引用块形态（如：左侧竖线颜色 + 粗细、背景色）
- [ ] 签名形式（如：GIF 图 + 细分割线、斜体文字签名）
- [ ] AI 声明卡（有 / 没有；如果有：背景色 + 图标形式）
- [ ] 出处块（有 / 没有；如果有：底色 + 字距 + 语气风格）
- [ ] 其他（画作引用标签、章节分隔符等）

### 段落习惯

- [ ] 是否段首 emoji（每段用 emoji 作视觉锚点）
- [ ] 是否首字下沉（lead paragraph 的第一个字放大浮动）
- [ ] 列表 marker 形态（实心点 / em-dash / 自定义符号）
- [ ] 有无章节编号前缀（如 `01 /`、`I.`、`§` 等）

## Worked example（示例博客「Demo 号」）

> 这是一个虚构示例，替换成你自己的。

### 基础排版

| 项 | 值 |
|---|---|
| 正文色 | `#222222` 深炭灰 |
| 行高 | `1.8` |
| 字距 | `0.5px` |
| 字号 | `16px` |
| 段距 | `margin-bottom: 10px` |
| 对齐 | `justify` |
| 边距 | `padding: 0 14px` |

### 招牌色

| 用途 | 值 |
|---|---|
| 主强调色（招牌蓝） | `#2F6FEB` |
| 次要文字 | `rgb(90, 90, 90)` |
| 分割线 | `#DDDDDD` |
| 卡片底色（AI 声明） | `#EEF2FA` |
| 出处块底色 | `#F5F5F5` |

### 标志性组件

1. **竖线小节标题**：左侧 3px 招牌蓝竖线 + 18px 无衬线标题（最具识别度的符号）
2. **段首图标**：每段首字前放小图标或 emoji 作视觉锚点
3. **灰底出处块**：`#F5F5F5` 底 + 12px 字 + 加宽字距 + `*注：` 谦逊语气开头
4. **AI 声明卡**：`#EEF2FA` 底 + 招牌蓝圆形 icon + 同色标题
5. **签名**：文末右对齐斜体文字 + 25% 宽细线

### 段落习惯

- 段首 emoji：是（每段开头放一个场景 emoji）
- 首字下沉：否
- 列表 marker：实心点（默认）
- 章节编号：无

---

## 用法

把你填好的 4 维 DNA 连同 1–3 篇旧文（或旧文截图）一起交给 `/new-theme` 的 classic 路径。classic 路径会在保留你现有视觉 DNA 的基础上，对每个组件做精细化打磨，输出一套新的 inline-styled HTML 主题供你直接使用。
